This privacy policy governs the Discord bots created by Acronical. My goal is to be transparent about the data I handle to provide a safe and functional experience on the Discord platform.

Data Collection

The bot processes User IDs, Server IDs, and message content strictly required for command execution and core functionality.

Data Usage

Data is used solely to respond to interactions. I never sell data or share it with third-party entities.

Security & Storage

Most data resides within Discord's infrastructure. Internal storage is restricted to secure databases containing only non-identifiable strings, IDs, and configuration settings required for persistence.
