Discord Bot Privacy Policy

This privacy policy is for the Discord bots that I have created and is hosted on the Discord platform. This privacy policy is to inform you about what data is collected and how it is used.

Data Collection

The bot collects data such as user IDs, server IDs, and message content. This data is used to provide the bot's functionality and to improve the user experience.

Data Usage

The data collected by the bot is used to provide the bot's functionality, such as responding to commands and interacting with users. The data is not shared with any third parties and is not used for any other purposes.

Data Storage

The data collected by the bot is stored securely on the Discord platform. In cases where data is not stored on the discord platform, it is stored in a secure database with only user ids, guild ids, and non-identifiable text or role ids.
